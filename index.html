Git&github

git init
Creates a local repository
git commit-m "message"
commits working tree to a brancmh
git workflow
Editing files in the working directory,adding files tothe staging area & saving changes to a git repository
git diff
checks diff btwn working directory and staging area
git log
used to vie commits in the repository
git add
adds files from the working directory to the staging area
git status
Inspects the content of the directories and staging areas
git branch -d
Deletes a local branch
git branch
adds a branch
git checkout -name
changes the working branch
git merge
merges current brancnh into main branch
git stash
allows you to get back to a clean commit point with a synchronized working tree and avoid losing local changes in the process
git stash pop
pops up work that was store using git stash
